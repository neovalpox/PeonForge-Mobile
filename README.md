# PeonForge Mobile

App Android compagnon pour [PeonForge](https://github.com/neovalpox/PeonForge).

Surveille tes sessions Claude Code depuis ton telephone, recois des notifications push, et vibecode a distance.

## Features

- **Suivi en temps reel** des sessions Claude actives
- **Terminal distant** avec boutons interactifs (Ctrl+C, Enter, y/n)
- **Notifications push** quand Claude finit une tache ou attend une permission
- **36 personnages WC3** deblocables par niveau avec portraits HD et voix FR
- **Tamagotchi** : nourris, caresse et entraine ton peon
- **Alliance vs Horde** : choisis ton camp
- **Connexion internet** via tunnel Cloudflare (4G/5G)
- **QR code pairing** pour se connecter au PC
- **Leaderboard** synchronise sur [peonforge.ch](https://peonforge.ch)

## Installation

### APK

Telecharge sur [peonforge.ch](https://peonforge.ch)

### Sources

```bash
git clone https://github.com/neovalpox/PeonForge-Mobile
cd PeonForge-Mobile
flutter run
```

Necessite Flutter 3.29+ et JDK 17.
